# rdzTTGOsonde Überblick

Diese Software macht aus einem ESP32-Board mit SX127x-LoRa-Chip und einem Display einen kleinen tragbaren Empfänger für Wettersonden (RS41, RS92, DFM06, DFM09, M10).
